# Icon and Calendar Redesign Plan

## 概要
ユーザーのフィードバックに基づき、以下の2点を改修します。
1. **サイドバーのアイコン変更**: サイドバー左上の紫色のグラデーションアイコン（`P`）を、Notion風のシンプルでモノクロームなアイコン（または絵文字）に変更します。
2. **カレンダー画面の改善**: カレンダー画面で「Gold」が最高の達成度であることが直感理にわかりにくい問題を解決します。

## 対象ファイル
- `components/layout/Sidebar.tsx`
- `app/calendar/page.tsx`
- `lib/utils.ts` (達成度のラベル・色定義、必要に応じて)

## 実装詳細
### 1. サイドバーアイコン
- `bg-gradient-to-br from-blue-500 to-purple-600` を削除。
- 代わりに白黒ベースのシンプルなアイコン（例: `Lucide` のアプリアイコン、または単に `📓` などの絵文字）に変更し、Notion全体のTech / プレーンな世界観に合わせます。

### 2. カレンダー画面の改善
- 達成度「Gold」「Silver」「Bronze」の視覚的な優位性を明確にします。
- 具体的には、Legend（凡例）の並び順を「Gold（最高）> Silver > Bronze」とし、ラベルに `👑 Gold (最高)` などのテキストを追加します。
- カレンダーセルのホバーツールチップやセル自体の表示にも、Goldの場合は特別なアイコン（王冠など）を表示するようにし、一目で最高の状態であることがわかるようにします。
